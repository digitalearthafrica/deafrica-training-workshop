Session 2: Datasets
=======================

This session is all about the datasets available through Digital Earth Africa. These datasets can be visualised through the Digital Earth Africa Map, or sorted by availability in the Digital Earth Africa Metadata Explorer. This session will cover how to use the Digitial Africa Sandbox to load datasets for a given area of interest and make a colour image.

Click on **Digital Earth Africa products** to get started. If you would like to review Session 1 content first, click **Session 1: Introduction** in the side menu.
